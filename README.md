# blockly_python
基于blockly 生成 python scrapy代码练手小项目

### Path

node.js  <= v10.16.3

管理员权限打开cmd执行：npm install --global --production windows-build-tools

### Run

git clone https://github.com/Zhui-CN/blockly_python.git

cd blockly_python

npm install

npm run dev
